## BUILD

启动进程文件

```pm2 start ecosystem.config.js```

停止进程文件

```pm2 stop ecosystem.config.js```

重启进程文件

```pm2 restart ecosystem.config.js```

重新加载进程文件

```pm2 reload ecosystem.config.js```

删除进程文件

```pm2 delete ecosystem.config.js```
